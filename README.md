# Designing a Static Management System Web Interface

## Description
This project is a static Student Management System interface created using HTML5 and CSS3.
It follows normal document flow and basic styling rules without using Flexbox, Grid,
Position, or Media Queries.

## Pages Included
- Home Page
- Login Page
- Dashboard Page
- Student List Page

## Technologies Used
- HTML5
- CSS3

## Folder Structure
project-folder/
- index.html
- login.html
- dashboard.html
- list.html
- style.css
